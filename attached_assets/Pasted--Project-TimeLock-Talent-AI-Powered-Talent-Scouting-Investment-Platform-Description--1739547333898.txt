### Project: TimeLock Talent - AI-Powered Talent Scouting & Investment Platform

**Description:**  
TimeLock Talent is a next-gen platform designed to revolutionize talent scouting and investment. Using **AI-powered ranking**, **blockchain tokenization**, and **transparent milestone tracking**, the platform allows **talents to tokenize their potential**, **investors to fund them**, and **all parties to track progress seamlessly**.

---

### Tech Stack:
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Node.js + Express.js + MongoDB (Mongoose)
- **Blockchain:** Solidity (ERC-20 token for Talent Tokens)
- **Authentication:** JWT (JSON Web Token)
- **State Management:** React Context API or Redux
- **Web3 Integration:** MetaMask & WalletConnect (for token investments)
